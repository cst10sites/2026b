External CSS

Internal and inline CSS define the look of a page (or a section of a page). In contrast, external CSS defines the rules for a collection of pages (i.e. website). External CSS uses a separate file containing the CSS and all pages within a website read from this file.

External CSS can also be used to reduce the amount of code inside an HTML file.

Embedding External Content

The most common way to embed external media is to use the iframe element. For YouTube videos, use the Share button followed by Embed to get the code. This method often produces an error in the validator. Also, not all videos can be embedded.
